JavaScript об'єктна мова. Масиви, функції, стрічки – це все об'єкти зі своїми методами та властивостями.
Можна створювати власні об'єкти.
var book = {
topic: “JavaScript”,
pages: 320
};
До полів обєкта book можна доступитися через . або [].
book.topic // “JavaScript”
book[“pages”] // 320
Також можна створити нові поля.
book.author = “Flanagan”; book.content = {} // пусте поле
Масив можна створити декількома способами:
  • Пустий масив з допомогою конструкора Array()
    var arr = new Array();
  • Пустий масив з заданим розміром
    var arr = new Array(10);
  • Заповнений масив через Array()
    var arr = new Array(“red”,”green”,true,1,2,3);
  • Заповнений масив через []
    var arr = [“red”,”green”, true,1,2,3);
Двовимірні масиви оголошуються як масив, в масиві.
var arr2 = [ [1,2,3], [4,5,6], [7,8,9] ];
Масиви мають наступні методи:
  • forEach() – обходить всі елемети масиву і виконує задану фінкцію.
  • map() – передає заданій функції елементи масиву і створює новий масив значень, що повернула функція.
  • filter() повертає підмножину заданого масиву.
  • every()повертає true якщо для всіх елеметів умова функції виконується.
  • some() повертає true якщо хоча б один елемент задовільняє умову функції.
  • indexOf() повертає позицію першого входження елемента в масив с початку.
  • lastIndexOf() повертає позицію першого входження елемента в масив з кінця.
Методи класу Array:
  • join() об'єднює всі елементи масиву у стрічку
  • reverse() міняє послідовність елементів на зворотню
  • sort() сортує масив в алфавітному порядку
  • concat() додає до масиву нові елементи
  • slice() повертає підмножину масиву в заданих межах, не змінюючи його
  • splice() повертає підмножину масиву в заданих межах,змінюючи його
  • push() pop() вставляє або вилучає елемент за принципом LIFO
  • toString() перетворює кожен елемент масива в стрічку

Функції – це параметризовані блоки програмного коду JavaScript, які можна викликати неодноразово.
Оголошення функції:

{блок виразів;}
function plus1(x)
{ return x+1; }
Функції можна присвоювати змінним:
var square = function(x)
{ return x*x; }
square(plus1(y) ); //виклик двох функцій
Функції можна прив'язати до об'єкта.
Метод - це функція, яка зберігається як властивість об'єкта.
Якщо існує функція f та об'єкт о, то можна створити метод цього об'єкта:
o.m = f;
Потім даний метод можна викликати.
o.m();
або
o["m"]();
У кожного метода та функції є своя лексична область видимості.
var a=1;
function go(){var a= 6;}
go();
alert(a);// a= 1;
