Мова Javascript

Javascript - це мульти-парадигмова мова програмування: об'єктно-орієтнована(прототипна), імперативна, функціональна, скриптова мова.
Особливості мови: наслідування обєктів відбувається прототипно, наявні функції першого класу, анонімні функції, змінним статично не надаються типи: тип змінної може змінюватися під час її використання відповідно до значення.

Обєктами в Javascript є форми, зображення, гіпертекстові посилання та інші компоненти web-сторінки, HTML-документ, що відображається у вікні браузера, вікно браузера і навіть сам браузер.

Основна ідея JavaScript – можливість опрацювання атрибутів HTML-контейнерів і властивостей середовища відображення в процесі перегляду HTML-сторінки користувачем. При цьому перезавантаження сторінки не відбувається.

Javascript можна використовувати щоб:
  • Змінювати елементи HTML
  • Видаляти елементи HTML
  • Створювати нові HTML елементи
  • Копіювати чи клонувати HTML елементи

Мова Javascript виникла у 1995 році. Розробник - компанія Netscape Communications, автор - Брендон Айк. Спочатку мова називалась Livescript. Ключові принципи були взяті з мов Self і Sheme. У 1996 Microsoft випутив аналог Javascript - JScript. Першим браузром, який підтримував цю реалізації був Internet Explorer 3.0.

Javascript базується на мові Ecmascript. На сьогодні всі сучасні браузери підтримують Esmacsript5.1. Остання версія Javascript - 1.8.5

BOM-об'єкти

Через BOM-об'єктів можна керувати поведінкою браузера за допомого Javascript.

  • Navigator
    navigator.userAgent - інформація про браузер.
    navigator.platform - інформація про платформу, відрізняє Windows/Linux/Mac..
    navigator.appVersion
    navigator.cookieEnabled
  • Screen
    screen.width/height/colordepth - інформація про екран.
  • Location
    Можна дізнатись будь-яку інформацію про поточний URL.
  • History
    Зберігає список відвіданих URL в цьому вікні браузера.
    history.length
    history.go()
  • Frame, timing, cookies
